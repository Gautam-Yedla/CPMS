# CPMS - Computerized Property Management System

A modular architecture separating frontend, backend, and machine learning services.

## Project Structure

- `Frontend/`: React + TypeScript (Vite)
- `Backend/`: Node.js + Express + TypeScript
- `Supabase/`: Database, Auth, and Security
- `Docs/`: Documentation and architecture diagrams

## Getting Started

### Frontend
1. `cd Frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd Backend`
2. `npm install`
3. `npm run dev`
